宿主环境

  小程序的运行环境分为逻辑层（App Service）和渲染层（View），其中 WXML 模板和 WXSS 样式工作在渲染层，JS 脚本工作在逻辑层。

API

  1. 获取用户的地理位置

 type: 'wgs84',
 success: (res) => {
   var latitude = res.latitude // 纬度
   var longitude = res.longitude // 经度
 }
})
